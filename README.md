# JoodKids (Next.js + Firebase + Cloudinary)

## تشغيل المشروع
1) ثبّت المتطلبات:
- Node.js 18+

2) داخل مجلد المشروع:
```bash
npm install
npm run dev
```

## إعداد Firebase
المشروع جاهز على Config الذي زوّدتني به داخل:
- `src/lib/firebase.ts`

## الأدمن (Admin UID)
تم تثبيت الأدمن كـ UID واحد فقط:
- `RFXkA9U7PeW6NkVYfFonufwq2Eg1`

## Cloudinary (Unsigned)
الإعدادات موجودة في:
- `src/lib/cloudinary.ts`

> مهم: Unsigned يسمح بالرفع من المتصفح. الحذف الكامل لصور Cloudinary يحتاج Cloud Function (سيرفر) لأن API Secret لا يوضع داخل المتصفح.

## Firestore Rules
ملف rules جاهز:
- `firestore.rules`

## ملاحظات
- استبدل ملفات الأيقونات في `public/` بملفات PNG صحيحة (192 و 512) + ضع شعارك الحقيقي في `public/logo.jpg`
- صفحة سياسة الدفع والاسترجاع موجودة: `/policy`
- العروض/الخصومات موجودة كـ `hasDiscount` و `discountPrice`


## استخدام احترافي
1) افتح المتجر واضغط شعار JoodKids 5 مرات → /admin
2) من لوحة التحكم افتح تبويب **المواسم/الأصناف**: أضف موسم ثم أضف أصناف داخله.
3) افتح تبويب **المنتجات**: عند كل منتج اختر الموسم والصنف ثم احفظ.
4) من القائمة الجانبية (☰) اختر الموسم ثم الصنف لفلترة المنتجات.

## الطلبات (Checkout)
- صفحة السلة: /cart ثم زر **إتمام الطلب** إلى /checkout.
- يتم حفظ الطلب في Firestore داخل collection اسمها **orders** ثم فتح واتساب برسالة جاهزة.
- لازم تنشر rules الموجودة في `firestore.rules`.

## تأكيد الطلب
بعد إرسال الطلب يتم تحويل العميل إلى `/order-success?orderId=...` لعرض رقم الطلب.

## لوحة تحكم الطلبات
داخل /admin → تبويب **الطلبات**: عرض + تغيير الحالة + تصدير Excel.

## تحسينات الطلبات
- بحث داخل الطلبات بالاسم/الهاتف/رقم الطلب.
- زر **نسخ رسالة واتساب** لكل طلب.
- زر **واتساب العميل** لفتح محادثة مباشرة.
